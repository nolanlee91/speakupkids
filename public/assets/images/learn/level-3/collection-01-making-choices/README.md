## Suggested lesson flow

1. Understand the shared goal and available choices.
2. Listen to two or more character opinions.
3. Match each opinion with its supporting reason.
4. Complete useful conversation turns.
5. Compare benefits and drawbacks.
6. Choose a response or compromise and justify it using structured options.
7. Complete a machine-graded Mini Check.

The image establishes the decision context only. Store all prompts, dialogue and answers in structured lesson data, never inside the artwork.
